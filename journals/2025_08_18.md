- [[Nix]]
	- rclone #google
		- Follow steps in https://rclone.org/drive/#making-your-own-client-id and create client_id/secret or use existing one in BitWarden
		- Run `rclone config`
		- `rclone mount gdrive: ~/gdrive --daemon -vfs-cache-mode writes`
	- [[VSCode]]
		- VScode is unable to find a keyring to store password https://code.visualstudio.com/docs/configure/settings-sync#_troubleshooting-keychain-issues
		- Go to `Preferences > Configure runtime arguments` then add the following
			- `"password-store": "gnome-keyring"`

- [[RPi]]
	- To set a static IP change `/etc/dhcpcd.conf`
	  ```
	  interface wlan0
	  static routers=192.168.20.1
	  static domain_name_servers=8.8.8.8
	  static ip_address=192.168.20.40/24
	  ```
	- Pi-Hole
		- Set up Static IP
		- `curl -sSL https://install.pi-hole.net | bash`
